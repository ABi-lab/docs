Class PaletteQuantizer

Allows the quantization of images pixels using web safe colors defined in the CSS Color Module Level 4. http://msdn.microsoft.com/en-us/library/aa479306.aspx Override this class to provide your own palette.

By default the quantizer uses FloydSteinberg dithering and the WebSafePalette
