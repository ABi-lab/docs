Apply<TPixel>(IImageProcessingContext<TPixel>, Action<Image<TPixel>>)

Applies the given operation to the mutable image. Useful when we need to extract information like Width/Height to parametrize the next operation working on the IImageProcessingContext<TPixel> chain. To achieve this the method actually implements an "inline" IImageProcessor<TPixel> with operation as it's processing logic.
