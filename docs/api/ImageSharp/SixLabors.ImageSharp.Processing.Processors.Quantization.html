Namespace SixLabors.ImageSharp.Processing.Processors.Quantization

Classes

FrameQuantizerBase<TPixel>

The base class for all IFrameQuantizer<TPixel> implementations

OctreeQuantizer

Allows the quantization of images pixels using Octrees. http://msdn.microsoft.com/en-us/library/aa479306.aspx

By default the quantizer uses FloydSteinberg dithering and a color palette of a maximum length of 255

PaletteQuantizer

Allows the quantization of images pixels using color palettes. Override this class to provide your own palette.

By default the quantizer uses FloydSteinberg dithering.

PaletteQuantizer<TPixel>

A generic palette quantizer.

QuantizedFrame<TPixel>

Represents a quantized image frame where the pixels indexed by a color palette.

WebSafePaletteQuantizer

A palette quantizer consisting of web safe colors as defined in the CSS Color Module Level 4.

WernerPaletteQuantizer

A palette quantizer consisting of colors as defined in the original second edition of Werner’s Nomenclature of Colours 1821. The hex codes were collected and defined by Nicholas Rougeux https://www.c82.net/werner

WuQuantizer

Allows the quantization of images pixels using Xiaolin Wu's Color Quantizer http://www.ece.mcmaster.ca/~xwu/cq.c

By default the quantizer uses FloydSteinberg dithering and a color palette of a maximum length of 255

Interfaces

IFrameQuantizer<TPixel>

Provides methods to allow the execution of the quantization process on an image frame.

IQuantizer

Provides methods for allowing quantization of images pixels with configurable dithering.
