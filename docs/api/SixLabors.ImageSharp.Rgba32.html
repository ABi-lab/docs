Struct Rgba32

Packed pixel type containing four 8-bit unsigned normalized values ranging from 0 to 255. The color components are stored in red, green, blue, and alpha order.

Ranges from <0, 0, 0, 0> to <1, 1, 1, 1> in vector form.

Implements
IPixel<Rgba32>
IEquatable<Rgba32>
IPackedVector<System.UInt32>
IPixel
Namespace: SixLabors.ImageSharp
Assembly: SixLabors.ImageSharp.dll
Syntax
public struct Rgba32 : IPixel<Rgba32>, IEquatable<Rgba32>, IPackedVector<uint>, IPixel
Remarks

This struct is fully mutable. This is done (against the guidelines) for the sake of performance, as it avoids the need to create new values for modification operations.
